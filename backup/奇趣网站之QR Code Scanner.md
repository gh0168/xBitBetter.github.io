### 二、如何使用 QR Code Scanner

#### 1. 访问网站 🌐

1. 打开您的设备上的任意浏览器（如 Chrome、Safari、Edge）。
2. 在地址栏输入 **https://qr-code-scanner.org/zh** 并按回车。
3. 网站加载后，您将看到简洁的主界面，包含“扫描二维码”和“上传图片”两个主要选项。

#### 2. 使用摄像头扫描二维码 📸

- **步骤**：
  1. 点击主界面上的“扫描二维码”按钮。
  2. 浏览器会请求访问设备的摄像头权限，点击“允许”。
  3. 将设备摄像头对准二维码，确保二维码完整出现在取景框内，距离约为 20-30 厘米。
  4. 扫描完成后，二维码内容（如链接、文本等）会立即显示在屏幕上。
- **小贴士**：
  - 确保光线充足，避免反光或阴影干扰扫描。
  - 如果扫描电子屏幕上的二维码（如电脑或手机屏幕），可适当降低屏幕亮度以提高识别率。[](https://www.qrscanner.org/zh-cn "")
  - 扫描结果可直接点击（如链接）或复制（如文本）使用。

#### 3. 上传图片扫描二维码 🖼️

- **步骤**：
  1. 在主界面选择“上传”选项。
  2. 点击后，系统会打开文件选择窗口。
  3. 从设备中选择包含二维码的图片（支持 JPG、PNG、GIF、SVG 等格式）。
  4. 上传完成后，工具会自动解码并显示二维码内容。
- **小贴士**：
  - 确保图片中的二维码清晰可见，避免模糊或部分遮挡。
  - 可直接拖放图片到网站界面，快速完成上传。
  - 适合扫描通过邮件、社交媒体接收的二维码图片。

#### 4. 查看和处理扫描结果 📋

- 扫描完成后，网站会显示二维码中的具体内容，例如：
  - **网址**：点击链接直接跳转到目标网站。
  - **联系信息**：可保存到设备联系人列表。
  - **Wi-Fi 凭据**：显示网络名称和密码，方便连接。
  - **文本**：可复制到剪贴板用于其他用途。
- 如果需要，可将结果保存或分享，部分场景支持直接导出为文本格式。

### 四、常见问题解答（FAQ）

**1. 需要下载应用程序吗？**  
不需要！QR Code Scanner 是纯网页工具，直接通过浏览器访问即可使用，无需安装任何软件。

**2. 可以扫描哪些类型的二维码？**  
支持所有标准二维码格式，包括网址、文本、联系信息、Wi-Fi 凭据、电子邮件、短信等。

**3. 如何处理扫描失败的情况？**

- 确保二维码清晰、无损坏或遮挡。
- 检查摄像头权限是否已开启。
- 调整光线或设备角度，保持二维码在取景框内。
- 如果是图片扫描，确认图片格式受支持（JPG、PNG 等）。

**4. 扫描二维码安全吗？**  
通常是安全的，但请谨慎扫描未知来源的二维码，以防跳转到恶意网站。建议在扫描前确认二维码来源可信。

**5. 是否需要互联网连接？**  
访问网站和打开链接需要联网，但图片扫描功能在上传后无需持续联网。
